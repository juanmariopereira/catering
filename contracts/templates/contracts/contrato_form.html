Seleccionar ubicación (Santa Cruz, Bolivia)

Mueve el pin en el mapa para elegir la ubicación. Al aceptar se llenarán dirección, enlace de Google Maps, latitud y longitud.

El mapa no pudo cargarse. Revisa en Google Cloud Console:
  • Facturación: el proyecto debe tener una cuenta de facturación asociada (hay crédito gratuito mensual).
  • APIs habilitadas: Maps JavaScript API y Places API.
  • Clave API: en tu .env define GOOGLE_MAPS_BROWSER_API_KEY (o GOOGLE_MAPS_API_KEY).
  • Restricciones: si la clave tiene restricción por “Sitios web”, agrega http://localhost:8000/* y http://127.0.0.1:8000/* para desarrollo, y tu dominio en producción.
